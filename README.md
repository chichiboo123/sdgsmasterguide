# 🌍 SDGs 마스터 수업 가이드

> **SDGs Master 앱**을 활용한 직소(Jigsaw) 팀 활동 수업 가이드 웹앱

[![GitHub Pages](https://img.shields.io/badge/배포-GitHub%20Pages-2F5E9E?style=flat-square&logo=github)](https://chichiboo123.github.io/sdgsmasterguide/)

---

## 소개

초등학생을 위한 SDGs(지속가능발전목표) 팀 활동 수업을 지원하는 가이드 웹앱입니다.  
학생들이 태블릿 PC로, 교사는 PC로 함께 사용하는 것을 전제로 설계되었습니다.

`<SDGs Master>` 앱으로 마스터 카드를 제작한 뒤, 직소 수업 모형에 따라 **역할별 전문가 모임**을 진행하고 패들렛에 공유하는 전 과정을 안내합니다.

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| **수업 흐름 안내** | 역할 나누기 → 앱 활동 → 카드 만들기 → 전문가 모임 → 패들렛 공유 → 전체 발표의 6단계 |
| **팀 역할 소개** | SDGs 탐색가 / 사례 해설가 / 디지털 전문가 / 카드 설명사 역할 설명 |
| **수업 슬라이드** | 수업 중 화면에 띄워 활동을 안내하는 6장의 가이드 슬라이드 |
| **전문가 모임 기록장** | 같은 역할끼리 모인 전문가 모임의 회의 내용을 역할별로 기록, 텍스트·이미지 복사 지원 |
| **패들렛 공유 안내** | 결과물 공유 방법 및 글 예시 제공 |
| **기록 자동 저장** | 입력 내용을 기기 로컬 스토리지에 자동 저장 |

---

## 수업 모형: 직소 (Jigsaw)

```
1. 팀 구성 & 역할 배정
       ↓
2. SDGs Master 앱으로 마스터 카드 제작
       ↓
3. 같은 역할끼리 전문가 모임 (기록장 활용)
       ↓
4. 원래 팀으로 복귀 & 패들렛 공유
       ↓
5. 카드 설명사 중심 전체 발표
```

## 사용 환경

- **학생**: 태블릿 PC (크롬 브라우저 권장)
- **교사**: PC 또는 전자칠판
- 별도 설치 없이 웹 브라우저에서 바로 사용 가능
